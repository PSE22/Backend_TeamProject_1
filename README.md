# 쇼핑몰 프로젝트 
---
Spring boot와 Vue.js를 활용한 쇼핑몰 웹 프로젝트입니다.

## 프로젝트 소개
---

## 주요 기능
---
- 검색창을 활용한 상품 검색 기능
- 위시리스트 및 장바구니 기능
- 상품/주문내역 관리
